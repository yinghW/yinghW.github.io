Learning Pointer as a Starter

Memory address

A pointer is the memory address of a variable.

The memory address is usually a hexadecimal and usually has 8 numbers.
0x at the start indicate that the number is hexadecimal.

0x0000B294

The address of a variable can be obtained by & , the address-of operator.

To access an address, *, the dereference operator, is used.

Dynamic variable

In C++, programmers must manually control creation and destruction of pointer variables with operators new and delete.

Arrays and dynamic arrays

So basically array is a pointer variable… that points to the position 0 of the array.

Pointers can be assigned as a array, but array cannot be assigned to a pointer.

Dynamic array allows us define the length of array based on user’s demand. When we use it, we need to create it, define it, and then delete it.

This is the full cycle of a dynamic array. The array itself doesn’t need to have a name. We just need to name its pointer.
